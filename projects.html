Full Stack Weather App

Simple Weather App that uses HTML, CSS, JavaScript and Node.js. The user inputs a location which is sent to the backend server (Hosted on Heroku). The server uses two 'OpenWeatherMap' API's which converts a location to data and returns said data to the user. if an incorrect location is entered, an error is returned. The result is a simple and useful weather app.
